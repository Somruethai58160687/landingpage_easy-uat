User acceptance testing

What is User acceptance testing?

UAT or User acceptance is the last step of sending a job to customer so the customer confirms the job exactly where they want and tests the job to find the error.Now,what do you have a evidence when customer test your job?

What is User acceptance testing document ?

We applied a template of test case which we learned from university to create template UAT for this website.This template is easy to understanding and not complexity for customer when they test your job.

What's in the UAT template?

Although,this template is not complete but it useful for your job.Some text you have to fill it by pen because something cannot be identified.Now,let to see the topic on the template.

Name

Name is a name of page which you fill when you create page

Date

Date is a date when customer test your jobt.

Conditions

At first,it doesn't have this topic but when we test this website we found a problem.Some pages have a condition that they must work on another page before they can work on this page.So that we have a this topic.

Tester

Tester is a name of person who test this page.

Test

Test is a step of test this page that you should to do.

Result

This topic we have two column for result of the test step and to confirm this step that pass or fail
